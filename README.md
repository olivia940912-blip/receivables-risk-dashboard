# 해외채권 리스크 진단 대시보드

"채권회수는 왜 지연되는가"를 여신한도·부보·연령·DSO 네 가지 각도에서 진단하는 인터랙티브 Plotly + Streamlit 대시보드.

- 데이터: 학습용 합성 데이터 (해외법인 채권잔액·여신한도/부보·대손충당금·팩토링·회수·매출 6종)
- 핵심 인사이트: 여러 차트를 가로질러 종합한 발견 5건을 근거→해석→시사점 구조로 최상단에 표시
- 관리 우선순위 워치리스트: 무보증초과·부보초과잔여리스크·91일이상 과반·회수지연 30일+·이중익스포저 5개 위험 신호를 합산한 TOP 10 회사 표
- 차트 6종: 부보여부별 연체비중 · 법인별 활용률×연체비중(단일 축 그룹 막대) · 연령구간별 대손충당금비율 · 법인별 연체율 · 부보기관별 연체율 · DSO×활용률 산점도
- 각 섹션마다 `[검증]` 캡션으로 verification/의 독립 재계산과 대조한 수치를 표시

## 로컬 실행

```
pip install -r requirements.txt
streamlit run app.py
```

## 폴더 구조

```
data/           원본 CSV 6종
charts/         차트 생성 스크립트, data_prep.py(공용 전처리) · theme.py(공용 색상/폰트)
charts/fonts/   나눔고딕(OFL) — matplotlib 참고용 차트 스크립트에서 사용
verification/   원본 CSV 독립 재계산 스크립트 + 검증결과.md
app.py          Streamlit 앱
```
